## SEO Checklist para Landing Pages

Para cada LP criada, garantir:

```
□ URL com keyword primária
□ H1 com keyword principal
□ 300-500 palavras de conteúdo (mín)
□ Keyword mencionada 3-5 vezes (natural)
□ Localização: "Campo Grande" mencionada 2-3x
□ Meta description com keyword
□ Alt tags em imagens
□ CTA acima do fold
□ Mobile responsivo
□ Velocidade < 3s
□ Schema: LocalBusiness + FAQPage
□ Link interno para homepage
```
